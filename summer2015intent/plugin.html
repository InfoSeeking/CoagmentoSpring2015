Exploratory Search User Study 2014: Firefox Plug in

Once you install the latest firefox browser, please install the following plugin (XPI).
Click here to install the plugin.
When Firefox prompts whether to allow for installation, click "Allow".
Then when it prompts you to install the plugin, click "Install Now".
Once installed and when prompted to restart, click "Restart Now".
Press 'CTRL+Shift+S'(for Windows users) or 'CMD+Shift+S'(for Mac users) in the browser to enable the side bar now.
